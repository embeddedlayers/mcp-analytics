## When to Use This

- Monthly or quarterly Shopify performance review
- Understanding what drives larger orders
- Identifying your highest-value customer segments
- Evaluating the impact of discounts on order value

## Data Format

Export from Shopify Admin → Orders → Export. The module works with the standard Shopify orders CSV format including columns like `Name`, `Total`, `Created at`, `Lineitem quantity`, `Discount Code`, etc.
